Form Input Attributes

We will discuss some attributes used in Input tag.

  • Read-only

    This attribute specifies that the input field is read only.


  • Disabled

    This attribute makes the input field unusable and unclickable.


  • Size

    This attribute specifies the length of the input field


  • Min

    This attribute specifies the minimum value for the input field. For example, this input field has minlength="5" and min="10".



  • Max

    This attribute specifies the maximum value for the input field. For example, this input field has maxlength="10" and max="100".



  • Placeholder

    This attribute displays a visual hint for the users.


  • Pattern

    This attribute sets a format for the input field and only accepts input in the specified format.


  • Multiple

    This attribute allows the user to enter multiple values in the input field.
